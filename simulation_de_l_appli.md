# Simulation du Parcours Utilisateur - Collector's Library

Voici comment naviguer dans votre application pour gérer votre collection :

### 1. Consulter votre collection
- **Écran de départ** : `{{DATA:SCREEN:SCREEN_16}}` (Ma Collection)
- C'est ici que vous voyez tous vos articles. Vous pouvez filtrer par type (Livres, DVD, CD, Manga) ou utiliser la barre de recherche.

### 2. Ajouter un nouvel article
- Depuis l'écran "Ma Collection", cliquez sur l'onglet **"Ajouter"** dans la barre de navigation ou sur le bouton **"+"** flottant.
- **Action** : Vous arrivez sur l'écran `{{DATA:SCREEN:SCREEN_2}}` (Scanner le Code-barres / Recherche manuelle).
- Vous pouvez scanner le code-barres de votre livre ou film pour l'ajouter instantanément.

### 3. Voir les détails d'un article
- Cliquez sur une carte d'article (ex: "Vagabond Vol. 1") dans votre collection.
- **Action** : L'écran `{{DATA:SCREEN:SCREEN_17}}` (Détails de l'article) s'affiche.
- Vous y trouverez le synopsis, l'éditeur, et vous pourrez marquer l'article comme "Lu" ou lui donner une note.

### 4. Gérer la sauvegarde Cloud (Google Drive)
- Ouvrez le menu latéral (icône menu en haut à gauche) : `{{DATA:SCREEN:SCREEN_10}}`.
- Cliquez sur l'option **"Paramètres de sauvegarde"**.
- **Action** : Vous accédez à `{{DATA:SCREEN:SCREEN_4}}` (Synchronisation et sauvegarde).
- Connectez votre compte Google et activez la sauvegarde automatique pour ne jamais perdre vos données.

### 5. Suivre vos statistiques
- Cliquez sur l'onglet **"Stats"** dans la barre de navigation.
- **Action** : L'écran `{{DATA:SCREEN:SCREEN_13}}` (Statistiques) vous montre la répartition de votre collection.
